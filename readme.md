# Agentic Email Assistant

An AI-powered email assistant that helps track job applications and recruiter conversations.

## Features
- Fetches job-related emails
- Tracks application pipeline stages
- Extracts deadlines and interview details
- Manages LinkedIn recruiter interactions
- Answers queries in natural language

## Tech Stack
- Python
- Gmail API
- Gemini LLM
- MongoDB (planned)

This project follows a clean, feature-branch Git workflow.
